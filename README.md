# phpcms新闻资讯系统开发
一个基于phpcms的仿新京网的新闻系统
后台登陆：http://本地地址/phpcms/admin.php     登陆名：phpcms  密码:phpcms

数据库的s配置信息如下：
路径：phpcms/caches/configs/database.php
return array (
	'default' => array (
		'hostname' => 'localhost',
		'port' => 3306,
		'database' => 'phpcms1',      数据库名称，可以自行tww修改
		'username' => 'root',         数据库管理员 root
		'password' => '',             数据库密码 默认为空
		'tablepre' => 'v9_',
		'charset' => 'utf8',
		'type' => 'mysqli',
		'debug' => true,
		'pconnect' => 0,
		'autoconnect' => 0
		),
);
